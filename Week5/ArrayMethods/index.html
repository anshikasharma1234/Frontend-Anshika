Arrays are lists of items. JavaScript gives us powerful methods to work with them!
Let's learn the 4 most important methods: map(), filter(), reduce(), forEach()

1Ô∏è‚É£ map() - Transform Each Item

üìù What it does: Creates a NEW array by transforming each item in the original array.

üîë Key Points:

  • Takes each item, does something to it, returns new value
  • Returns a NEW array (same length as original)
  • Does NOT change the original array
  • Perfect for: Converting data, calculations on each item

Example: Double each number

// Original array
let numbers = [1, 2, 3, 4, 5];

// Using map() to double each number
let doubled = numbers.map(function(num) {
  return num * 2;
});

// Result: [2, 4, 6, 8, 10]

How it works step-by-step:

// map() goes through each item:
// Round 1: num = 1, returns 1 * 2 = 2
// Round 2: num = 2, returns 2 * 2 = 4
// Round 3: num = 3, returns 3 * 2 = 6
// Round 4: num = 4, returns 4 * 2 = 8
// Round 5: num = 5, returns 5 * 2 = 10
// New array: [2, 4, 6, 8, 10]

2Ô∏è‚É£ filter() - Select Specific Items

üìù What it does: Creates a NEW array with only items that pass a test (condition).

üîë Key Points:

  • Tests each item with a condition (true/false)
  • Returns NEW array with only items that passed (true)
  • Result can be shorter than original array
  • Perfect for: Finding specific items, removing unwanted data

Example: Get only even numbers

// Original array
let numbers = [1, 2, 3, 4, 5, 6, 7, 8];

// Using filter() to get even numbers only
let evenNumbers = numbers.filter(function(num) {
  return num % 2 === 0;
});

// Result: [2, 4, 6, 8]

How it works step-by-step:

// filter() checks each item:
// num = 1, is 1 % 2 === 0? NO (false) ‚Üí skip
// num = 2, is 2 % 2 === 0? YES (true) ‚Üí keep
// num = 3, is 3 % 2 === 0? NO (false) ‚Üí skip
// num = 4, is 4 % 2 === 0? YES (true) ‚Üí keep
// ... and so on
// New array: [2, 4, 6, 8]

3Ô∏è‚É£ reduce() - Combine Into Single Value

üìù What it does: Reduces an array to a SINGLE value by combining all items together.

üîë Key Points:

  • Takes all items and combines them into ONE value
  • Has an "accumulator" that stores the running total
  • Returns a single value (number, string, object, etc.)
  • Perfect for: Summing, counting, combining data

Example: Sum all numbers

// Original array
let numbers = [1, 2, 3, 4, 5];

// Using reduce() to sum all numbers
let sum = numbers.reduce(function(total, num) {
  return total + num;
}, 0); // 0 is the starting value

// Result: 15

How it works step-by-step:

// reduce() combines items:
// Start: total = 0 (starting value)
// Round 1: total = 0, num = 1, returns 0 + 1 = 1
// Round 2: total = 1, num = 2, returns 1 + 2 = 3
// Round 3: total = 3, num = 3, returns 3 + 3 = 6
// Round 4: total = 6, num = 4, returns 6 + 4 = 10
// Round 5: total = 10, num = 5, returns 10 + 5 = 15
// Final result: 15

4Ô∏è‚É£ forEach() - Do Something With Each Item

üìù What it does: Runs a function for each item in the array. Does NOT return anything.

üîë Key Points:

  • Loops through each item in the array
  • Does NOT return a new array
  • Just performs an action (like printing, updating, etc.)
  • Perfect for: Displaying data, side effects, logging

Example: Print each item

// Original array
let fruits = ["Apple", "Banana", "Orange"];

// Using forEach() to print each fruit
fruits.forEach(function(fruit, index) {
  console.log(index + ": " + fruit);
});

// Console output:
// 0: Apple
// 1: Banana
// 2: Orange

How it works step-by-step:

// forEach() visits each item:
// Round 1: fruit = "Apple", index = 0 ‚Üí prints "0: Apple"
// Round 2: fruit = "Banana", index = 1 ‚Üí prints "1: Banana"
// Round 3: fruit = "Orange", index = 2 ‚Üí prints "2: Orange"
// Returns: nothing (undefined)
