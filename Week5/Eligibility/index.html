Method 1Ô∏è‚É£: If-Else Statement

What it does: Checks a condition and runs different code based on whether it's true or false.

Structure:

if (condition) {
  // Code runs if condition is TRUE
} else {
  // Code runs if condition is FALSE
}

Example:

if (age >= 18) {
  message = "You are eligible!";
} else {
  message = "Sorry, you are not eligible.";
}

Best for: When you have multiple lines of code to run, or complex logic.

Method 2Ô∏è‚É£: Ternary Operator

What it does: A shortcut for simple if-else statements. It's a one-liner!

Structure:

condition ? valueIfTrue : valueIfFalse

Example:

message = (age >= 18) ? "You are eligible!" : "Sorry, you are not eligible.";

Best for: Simple decisions with one line of code for each outcome.
